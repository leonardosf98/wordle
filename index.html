WORDLE

O objetivo desse jogo é descobrir a palavra certa, você tem 6 tentativas para isso!


Atualmente, este jogo só funciona com palavras em inglês. Caso você digite uma palavra que não é válida, a palavra sumirá.


Se o quadrado ficar verde, a letra está correta no lugar correto, caso apareça amarelo, a letra tem na palavra mas está no lugar errado, e caso seja vermelho, não existe a letra na palavra.
